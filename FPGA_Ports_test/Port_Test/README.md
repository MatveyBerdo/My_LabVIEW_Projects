# FPGA Ports Test / Port_Test

Кластер для тестирования работоспособности портов ввода/вывода FPGA на базе модуля **PXI-7951R (FlexRIO)** и адаптера **NI 6581**.

Проект предназначен для быстрой проверки целостности цифровых линий ввода/вывода, диагностики оборудования и верификации правильности подключения внешних устройств к адаптеру.

## Оборудование

Проект разработан и протестирован с использованием следующего оборудования:

*   **PXI-7951R**: Модуль FlexRIO на базе FPGA (Xilinx Virtex-5). Позволяет реализовать пользовательскую логику с жесткими требованиями по времени (real-time) и параллельной обработке сигналов.
*   **NI 6581**: Модуль согласования сигналов (адаптер) для PXI-7951R. Предоставляет набор цифровых линий ввода/вывода и подключается непосредственно к передней панели FPGA-модуля.
*   **PXI-4110**: Трехканальный программируемый источник питания постоянного тока. Используется для подачи питающих напряжений на тестируемые цепи или внешние устройства при проведении расширенной диагностики.
