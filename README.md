# Учебные проекты LabVIEW FPGA (PXI-7951R + NI 6581)

Этот репозиторий содержит учебные проекты, созданные в среде LabVIEW для изучения работы с модулями PXI и FPGA технологии NI FlexRIO.

## Оборудование

Основное целевое оборудование, используемое в проектах:
*   **PXI-7951R**: Модуль FlexRIO с FPGA (Xilinx Virtex-5) для разработки пользовательской логики.
*   **NI 6581**: Модуль согласования сигналов (адаптер) для PXI-7951R с цифровыми линиями ввода/вывода.
*   **PXI-4110**: Трехканальный программируемый источник питания постоянного тока.

## Структура репозитория

Репозиторий организован по папкам, каждая из которых содержит отдельное направление учебных работ:

*   **[Projects_study/](Projects_study/)**:
    Здесь собраны общеучебные проекты LabVIEW. В данную папку входят проекты, не связанные с FPGA (Host-проекты), включая работу с прибором **PXI-4110** (управление источником питания).

*   **[FPGA_projects_study/](FPGA_projects_study/)**:
    Проекты, посвященные программированию FPGA на модуле PXI-7951R. Изучается логика работы с цифровыми линиями через адаптер NI 6581.

*   **[FPGA_Ports_test/Port_Test](FPGA_Ports_test/Port_Test)**:
    Экспериментальный кластер для тестирования портов ввода/вывода FPGA (без использования PFI линий). Создан недавно для отработки базовых операций.
